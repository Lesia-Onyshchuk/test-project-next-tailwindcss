# Test Project - Next.js & Tailwind CSS

A simple landing page built with **Next.js** and **Tailwind CSS**, deployed on **Vercel**.

## Features

- Responsive design
- Tailwind CSS styling
- Next.js fast performance

## Installation

```bash
git clone https://github.com/yourusername/test-project-next-tailwindcss.git
cd test-project-next-tailwindcss
npm install
npm run dev
```

## Usage

Open your browser at http://localhost:3000 to view the app.

## Deployment

Deployed on Vercel: https://test-project-next-tailwindcss.vercel.app
